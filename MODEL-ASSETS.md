# Model Assets

These files are part of the static Pocket TTS payload used during setup:

- `pocket-tts-model-minimal.json`
- `model.safetensors`
- `tokenizer.model`
- `alba.safetensors`
- `anna.safetensors`
- `michael.safetensors`

The manifest file is visible in this repo so users can see exactly what setup downloads.

Some of the largest binary files are hosted as static download assets referenced by the manifest because GitHub repo file-size limits prevent tracking the biggest model file directly in the repo.
